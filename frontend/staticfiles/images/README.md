# Logo AGCF Voyages

## Emplacement du logo

Placez votre fichier logo ici avec le nom : **`logo-agcf-voyages.png`**

## Formats supportés

- PNG (recommandé avec fond transparent)
- JPG/JPEG
- SVG

## Taille recommandée

- **Navbar (barre de navigation)** : 40px de hauteur
- **Page d'accueil** : 60px de hauteur
- **Footer** : 30px de hauteur
- **Aperçu billet** : 35px de hauteur

## Structure du fichier

```
static/
└── images/
    └── logo-agcf-voyages.png  ← Placez votre logo ici
```

## Après avoir ajouté le logo

1. Placez votre fichier logo dans ce dossier (`static/images/`)
2. Nommez-le exactement : `logo-agcf-voyages.png`
3. Le logo s'affichera automatiquement dans :
   - La barre de navigation
   - La page d'accueil
   - Le footer
   - L'aperçu du billet

## Note

Si le logo n'est pas trouvé, l'icône de train par défaut sera affichée à la place.
